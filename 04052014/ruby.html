Why are we talking about Ruby?

Ruby is the programming language

Rails is the framework (a Ruby application).

Why are we talking about Ruby when this is a Intro to Rails course? Rails wouldn't exist without Ruby. The Rails framework is written in Ruby and provides a lot of features that makes web development easier. It's possible to write web applications in just plain Ruby, but it wouldn't be much fun.

What is Ruby?

Programming language created by Yukihiro "Matz" Matsumoto. First publicly released in 1995.

Ruby Design Goals

"I hope to see Ruby help every programmer in the world to be productive, and to enjoy programming, and to be happy. That is the primary purpose of Ruby language." - Matz


  • Open Source
  • Object oriented
  • Dynamic
  • Loosely typed

Open Source

Anyone can access the source code to read, contribute, or fork to hack with on their own. Ruby Source at Github

Object Oriented

Everything is an object in Ruby. What does this mean?

It means that everything responds to "messages" - how you access attributes and functions of objects

Dynamic

The language can be modified or extended at runtime. 


	class Numeric
	def plus(x)
	self.+(x)
	end
	end

	y = 5.plus 6
	# y is now equal to 11

Ruby docs for Numeric

Loosely Typed

You don't need to tell Ruby what "type" a variable needs to be. 


	name = "Ben"
	name.class
	=> String
      
	name = 10
	name.class
	=> Fixnum
